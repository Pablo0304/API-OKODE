Arquitectura

La arquitectura HTML de este proyecto consta de 3 ventanas, una para la lista de películas, otra para ver más detalles de la película seleccionada y la que está observando que explica cómo se ha creado el proyecto.

Ventana Lista

  • Con: se consigue que si no hay películas en el objeto donde están guardadas para mostrarse, se visualice la imagen de OKODE Digital Transformation.
  • Con: se consigue recorrer mediante un bucle todas las películas que nos ofrece la BD y así se van mostrando en la base de datos.

Ventana Detalle

  • Con: se consigue que si se recarga la página y se pierden los datos de la película seleccionada no se muestre nada, para solucionar los errores de visionado del documento
